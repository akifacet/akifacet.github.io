Saga-Orchestrated Distributed Order System

Java ve Spring Boot ile geliştirdiğim event-driven mikroservis sipariş sistemi. Apache Kafka üzerinden haberleşir. Orchestrated SAGA ile distributed transaction yönetimi, Transactional Outbox ile güvenilir event publish, idempotency ile duplicate message dayanıklılığı hedeflenmiştir. Docker ile containerize edilmiş, AWS ve Kubernetes ortamlarına uygun şekilde tasarlanmıştır.
